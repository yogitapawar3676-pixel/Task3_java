##  Overview
This project is a simple **Library Management System** built using **Object-Oriented Programming (OOP)** in Java.

## Features
- Add books and users  
- Issue book to a user  
- Return issued book  
- Display all books with status  
- Demonstrates OOP concepts

  - Encapsulation  
  - Inheritance  
  - Polymorphism  

##  Technologies Used

- Terminal / CMD
